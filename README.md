# 概要

Angular でivyを使用したサンプルです。  
https://angular.io/guide/ivy

# 実行環境

* Node.js - 10.x
* Yarn - 1.12.x

# 使用ライブラリ

* Angular - 8.0.x

# 動作確認

## 1. サンプルのダウンロード

```
git clone git@github.com:yasu-s/ng-ivy-sample.git
```

## 2. パッケージインストール  

```
cd ng-ivy-sample
yarn
```

## 3. サンプルの起動  

```
yarn start
```
